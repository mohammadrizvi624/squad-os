# Marketplace Experience — Squad OS

Scaling isn't about making yourself faster - it's about making your team better.

This repo is the **Squad OS** for the Marketplace Experience squad at Lightspeed: the shared knowledge base that turns our collective context into something AI can actually use. Product docs, strategy and OKRs, research, competitive intel, engineering and analytics — structured so that when anyone starts a new AI session, the agent already knows what we're working on.

Instead of one person bottlenecking context, everyone on the squad can self-serve. The repo becomes the shared brain of the team.

**Start here:** [`CLAUDE.md`](CLAUDE.md) (home base) and [`product-development/product/product-context/marketplace-overview.md`](product-development/product/product-context/marketplace-overview.md) (product primer).
