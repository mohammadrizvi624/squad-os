# Data Engineering

A **light footprint** for data-pipeline work the squad is directly involved in.
Platform-level data engineering across the Wholesale Network is owned by
**Data Foundations** (Connor Courtney's team) — this folder is for shared or
squad-specific pipeline plans and design proposals, not the whole platform.

## Folders

| Folder | What's Here |
|--------|-------------|
| `plans/` | Data pipeline implementation plans |
| `rfcs/` | Data pipeline design proposals |

Organized **flat**. Tie artifacts to features via `../feature-index.yaml`.

## Naming Conventions

- **Plans:** `{pipeline-name}.md`
- **RFCs:** `{pipeline-name}-rfc.md`
